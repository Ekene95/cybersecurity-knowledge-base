# Learning Resources

Curated list of high-quality, free Linux learning materials suitable for beginners and security professionals.

## Free Online Courses

### 1. Linux Journey
**URL:** [https://linuxjourney.com/](https://linuxjourney.com/)

**Description:** Offers a completely free, structured path for learning Linux from scratch. Designed for all levels, making it beginner-friendly.

**Topics covered:**
- Command line basics
- Text manipulation
- User management
- Permissions
- Processes
- Package management
- Networking

**Best for:** Complete beginners who want a structured learning path

---

### 2. edX - Introduction to Linux
**URL:** [Linux Foundation's "Introduction to Linux"](https://www.edx.org/course/introduction-to-linux)

**Description:** Free to audit course from The Linux Foundation. Provides a solid foundation in Linux basics, perfect for beginners with no tech background.

**What you'll learn:**
- Linux philosophy and concepts
- Graphical interface and command line
- System configuration
- Common applications
- Command line operations

**Best for:** Those who prefer video lectures and structured curriculum

---

### 3. LearnLinux.tv
**URL:** [https://www.learnlinux.tv/](https://www.learnlinux.tv/)

**Description:** Free beginner tutorials with detailed step-by-step video lessons on YouTube.

**YouTube Channel:** [NetworkChuck](https://www.youtube.com/c/NetworkChuck)

**Features:**
- Practical, hands-on approach
- Fun and engaging presentation
- Real-world scenarios
- Beginner-friendly exercises

**Best for:** Visual learners who prefer YouTube content

---

### 4. The Linux Documentation Project (TLDP)
**URL:** [https://www.tldp.org/LDP/intro-linux/html/](https://www.tldp.org/LDP/intro-linux/html/)

**Description:** Free, comprehensive, well-written guides excellent for beginners who want to understand the fundamentals.

**Content includes:**
- Detailed HOWTOs
- Guides and manuals
- FAQs
- In-depth technical documentation

**Best for:** Readers who prefer documentation over videos

---

- Run Linux directly on Windows 10/11
- No VM overhead
- Great for developers

## Learning Path Suggestion

### Complete Beginner
1. Start with **Linux Journey** (basics)
2. Watch **NetworkChuck** videos for motivation
3. Practice on **Linux Survival** interactive platform
4. Read **TLDP** guides for deeper understanding

### Security Professional
1. Complete **edX Introduction to Linux**
2. Work through **OverTheWire: Bandit** challenges
3. Setup **Kali Linux** VM
4. Practice on **TryHackMe** Linux rooms
5. Reference this knowledge base for specific commands

### System Administrator
1. **edX Introduction to Linux** course
2. **TLCL (The Linux Command Line)** book
3. Setup home lab with VirtualBox
4. Practice with **LearnLinux.tv** tutorials
5. Study **Linux From Scratch** for deep knowledge

## Tips for Effective Learning

1. **Practice regularly** - Use Linux as your daily driver if possible
2. **Break things** - Learn by fixing your mistakes in a VM
3. **Use man pages** - Get comfortable with `man command`
4. **Script everything** - Automate repetitive tasks
5. **Join communities** - Ask questions, help others
6. **Work on projects** - Build real things
7. **Read logs** - Understanding errors is crucial
8. **Study security** - Learn Linux from attacker's perspective
